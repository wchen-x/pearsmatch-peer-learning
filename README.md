# PearsMatch

PearsMatch is a full-stack peer-learning platform that matches students based on complementary academic and career skills.

Users create profiles, list skills they can teach and skills they want to learn, receive match suggestions, send connection requests, and chat after matching.

## Planned Tech Stack

- React
- TypeScript
- Tailwind CSS
- Spring Boot
- PostgreSQL
- JWT Authentication

## MVP Features

- User registration and login
- Student profile creation
- Teach and learn skill lists
- Match suggestions based on complementary skills
- Match requests
- Basic chat after matching
- Block/report safety feature
